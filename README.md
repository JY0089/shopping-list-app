# 쇼핑 리스트 앱

 HTML, CSS, JavaScript로 만든 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 아이템 체크/체크 해제
- 아이템 개별 삭제
- 완료 항목 일괄 삭제
- localStorage를 통한 데이터 유지 (새로고침 후에도 데이터 보존)

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 테스트

Playwright를 사용한 자동화 테스트가 포함되어 있습니다.

```bash
npm install
npm test
```
